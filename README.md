# kaminskakatarzyna.com

Strona internetowa Katarzyny Kamińskiej — Autoryzowany przedstawiciel Thermomix®

## Jak uruchomić lokalnie

```bash
npm install
npm run dev
```

Strona będzie dostępna pod `http://localhost:5173`

## Konfiguracja

Wszystkie dane kontaktowe i treści znajdują się w obiekcie `CONFIG` na górze pliku `src/App.jsx`. Zmień dane tam — reszta strony zaktualizuje się automatycznie.

## Deploy

Strona jest automatycznie deployowana na Vercel po każdym pushu do brancha `main`.
